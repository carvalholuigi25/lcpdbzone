# List of commands for project

## Usage: start npm with command name separated

| Title | Description | Status |
| :----: | :----: | :----: |
| start | It will start and watch the project (in production environment) | :white_check_mark: |
| startdev | It will start and watch the project (in development environment) | :white_check_mark: |
| startmyapi | It will start and watch the LCP api | :white_check_mark: |
| startmyapi2w | It will start and watch my custom api nodejs server (using by nodemon) | :white_check_mark: |
| startmyapi2 | It will start and watch my custom api nodejs server | :white_check_mark: |
| startapis | It will start and watch all apis | :white_check_mark: |
| startall | It will start and watch all apis and project (in production environment) | :white_check_mark: |
| startalld | It will start and watch all apis and project (in development environment) | :white_check_mark: |
| starttestall | It will start and watch the LCP api and testing the project (in development environment) | :white_check_mark: |
| starttestcovall | It will start and watch the LCP api and testing the project and output the test of code coverages (in development environment) | :white_check_mark: |
| cleancache | It will clear the cache of installed dependencies for project | :white_check_mark: |
| verifycache | It will verify and clear the cache (if exist) of installed dependencies for project | :white_check_mark: |
| delnmodules | It will delete node_modules | :white_check_mark: |
| delpacklock | It will delete package-lock.json | :white_check_mark: |
| build | It will build the project | :white_check_mark: |
| watch | It will build the project while watching it in development environment | :white_check_mark: |
| test | It will start the unit tests for project | :white_check_mark: |
| testcoverage | It will start the tests of code coverage for project | :white_check_mark: |
